# Sprint Stratégie de Marque

App de révision interactive (un seul fichier HTML, autonome) pour réviser la Stratégie de Marque : parcours en 10 modules, schémas SVG, flashcards et quiz noté.

👉 **Voir le site** : activé via GitHub Pages (onglet *Settings → Pages*).

Ouvre simplement `index.html` dans un navigateur — aucune dépendance.
